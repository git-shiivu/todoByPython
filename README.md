
# To-Dos by Python

It is a Python program that is use for making To-Dos, it can perform CRUD Operations to work, and it stores data into the “todo.txt” file. We can access it using Command Line Interface (CLI).

## How to use it?

### Step-1, Installation:
- Install Python interpreter from a Authentic website.
- Note - I am using Python version - 3.14.0

### Step-2, Download the program locally:

#### If you have "git":
- Perform this Command: ``git clone https://github.com/git-shiivu/todoByPython.git``

#### If you have not:
- Go to the "Code" button and left click.
- Go to the "Download ZIP" Option and select your desired file location.

### Step-3, Run the program:
- Open the Terminal/Command Prompt of your Device.
- Go to the folder using Terminal/Command Prompt command, where you have downloaded the program.
- Perform this Command: ``python3 todo.py``
